# 🐍 Classic Snake Game

A fun, classic Snake game built with **HTML5**, **CSS**, and **JavaScript**. Play it directly in your browser!

## How to Play

- **Arrow Keys** or **WASD** → Move the snake
- Eat the **red food** to grow and score points
- Avoid hitting the **walls** or your **own tail**
- **Spacebar** → Pause / Resume
- **R** key → Restart after game over

## Getting Started

Simply open the `index.html` file in any modern web browser. No installation or setup required!

## Features
- Smooth grid-based gameplay
- Score tracking
- Pause functionality
- Clean game-over screen
- Keyboard controls
